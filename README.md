# Shipment-Details

Steps to run the project:

1) <code>yarn</code> or <code>npm install</code>
2) <code>yarn dev</code> or  <code>npm run dev</code>

<h3>Technology Stack:</h3>

1) JSON Server
2) REACT JS
3) Jest and Enzyme
4) HTML
5) CSS
6) JavaScript
